# Soulmate-website-2
this is a second approach of the soul mate webiste, still working on updates
📁 Forever-Begins-Here/
   ├── index.html          ← The main page
   ├── style.css           ← All styles & animations
   ├── script.js           ← All interactivity
   ├── parchment.png       ← Your uploaded parchment image
   ├── 01.jpg - 09.jpg    ← Your 9 memory photos
   └── thousands-years.mp3 ← Your uploaded music
   | Feature                 | Description                                                                                                  |
| ----------------------- | ------------------------------------------------------------------------------------------------------------ |
| **6 Cinematic Scenes**  | Awakening → Locked Letter → Memory Garden → Wish → Promise → Grand Finale                                    |
| **Password Protection** | Type `lovers` to unlock the secret letter                                                                    |
| **Parchment Letter**    | Written on your uploaded `parchment.png` with burned edges, wax seal, ink animation                          |
| **9 Memory Photos**     | Each with a unique transition (fog, petals, particles, ink, butterflies, stars, rain, burn, zoom)            |
| **Atmospheric Effects** | Twinkling stars, glowing moon, drifting clouds, fog, fireflies, rose petals, dandelion seeds, shooting stars |
| **Music Box**           | Click to play `thousands-years.mp3`                                                                          |
| **Parallax**            | Mouse movement shifts the moon & clouds                                                                      |
| **Navigation Dots**     | Click to jump between scenes                                                                                 |
| **Keyboard Nav**        | Arrow keys to navigate                                                                                       |
| **"Relive" Button**     | Restarts the entire experience                                                                               |
